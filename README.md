# Stooq reader

### Setup

requirements:
* java
* maven
* bower

Install:
```
mvn clean install
```
```
cd src/main/resources/static/
```
```
bower install
```
### Run

Inside /src directory type:
```
mvn spring-boot:run
```

Visit http://localhost:3333/
